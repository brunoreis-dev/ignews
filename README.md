# Rocketseat - Ignews

## Clonando o projeto

`| git clone git@github.com:brunoreis-dev/ignews.git |`

## Instalação

`| yarn install |`

## Executando o projeto

Abaixo os comandos que são usados no projeto:

| Comando | Description |

| ----------------------- | --------------------------------------------------------------------------------------------- |

`| yarn install |` Instala as dependências do projeto |

`| yarn dev |` Executa o projeto em modo de desenvolvimento na porta `3000` |

## Padrões de código/desenvolvimento

- Para nomenclatura de branches, utilizamos o padrão do Gitflow

  - `feature/${nomeDaFeature}` - Para features novas
  - `hotfix/${nomeDoHotfix}` - Para hotfixes

### Technologys:
- Next.Js
- React
- Sass
- Typescript
- Jest


### Features:
- [x] Autenticação de usuário via OAtuh GitHub;
- [x] Assinatura para leitura dos posts;
- [x] Pagamento via Stripe;
- [x] Posts dinâmicos vindo do CMS Prismic;
- [x] Testes unitários;
